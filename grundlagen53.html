Bildung und Verwendung passiver Partizipien Präsens

Diese Partizipien beschreiben Handlungen, die im Moment an einem Objekt vollzogen werden. Sie werden nur von transitiven Verben im unvollendeten Aspekt (НСВ) gebildet.

  • 1. Konjugation: мы-Form + Endung (Suffix -ем-). Beispiel: читать → читаем → читаемый.
  • 2. Konjugation: wir-Form + Endung (Suffix -им-). Beispiel: любить → любим → любимый.
  • Ausnahme: Verben auf -авать behalten das -ва-. Beispiel: давать → даваемый.
  • Der Handelnde steht im Instrumental (5. Fall).
